# react-widget-browser

### Commands
command | Description
--- | ---
**yarn** | install dependency
**yarn start** | run dev mode
**yarn build** | build browser compatible into /dist -> react-widget-browser.min.js
**yarn serve:dist** | serve the /dist into localhost:5000
